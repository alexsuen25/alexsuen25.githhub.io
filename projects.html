Project: Chatbot
Duration: October 28th, 2022 - November 7th, 2022, approximately 1/2 month
Course: AP Computer Science Applications
Type of Project: Group

For the Chatbot Project, the goal of this project was to create a chatbot that would be able to converse with a user and have a certain bias. The instructions on what should be included in the project were given on Runestone. My partners and I began to work on the project by looking at all the possible libraries and seeing which ones could possibly contribute to the project most efficiently. We listed our ideas on this spreadsheet and put all of our documentation on this presentation. We dug further into each of the libraries that we chose to see if there was anything off about it. For the coding of the project, we used Python ad using the PyTorch as the natural language processing platform. My role was to create the class of the chabot, filter it through the pipeline, and make sure that it would work properly. In the presentation, we showcased our documentation, how we created the project, and what extra features we implemented in the project. Through this activity, I was able to develop my critical thinking skills and researching skills because I had to identify the problems in the files and be able to fix them. 
Here is the link to my documentation for this project.
